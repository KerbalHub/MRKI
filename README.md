# MRKI
A Kerbol System Revamp mod

## MRKI INSTALLATION INSTRUCTIONS:
1. Install dependencies: Kopernicus, KopernicusExpansion, 000_NiakoUtils
2. Drag-and-drop MRKI folder into GameData
3. Semi-Dependencies: Sigma Dimensions (Rescale), Kronometer
4. Recommended mods: Parallax Continued (Delete Parallax_StockPlanetTextures), EVE (Keep StockVolumetricClouds), Scatterer, Deferred+Shabby
